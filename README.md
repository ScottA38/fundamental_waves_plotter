A program designed to take a json-formatted data structure comprising of
note information for a specific peice of music or audio.
See the supplied JSON file in this repo for the JSON data structure defintion.
 
With a recognised JSON data strucuture processes this data into sine waves at the correct positions on an x-
axis,t. The sin waves will be summed together into one single amplitude wave to try and mimic wave interference: https://en.wikipedia.org/wiki/Wave_interference
